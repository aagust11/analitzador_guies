# Analitzador de guies

Aplicació SPA construïda amb React + TypeScript + Vite per analitzar guies institucionals d'IA en educació. Ara mateix només conté l'esquelet de navegació (Pas 1 del pla).

## Scripts disponibles

Després d'instal·lar dependències amb `npm install`:

- `npm run dev`: inicia el servidor de desenvolupament de Vite.
- `npm run build`: executa la compilació de TypeScript i genera els fitxers de producció.
- `npm run preview`: previsualitza la compilació generada.

## Estructura actual

- Barra lateral amb enllaços a Guies, Graella comparativa i Configuració.
- Top bar amb el títol contextual.
- Pàgines bàsiques (`Home`, `AnalysisWorkspace`, `ComparativeGrid`, `Settings`) amb contingut placeholder.

En els propers passos s'anirà afegint la capa de dades, persistència local i funcionalitats d'anàlisi.
